# GitHub Pages Upload Guide

Upload the CONTENTS of this folder directly to the root of the `portfolio` repository.

Correct structure:

```text
portfolio/
├── index.html
├── style.css
├── script.js
├── profile.json
├── amazon-recommendation-system.html
├── airline-revenue-optimization.html
├── .nojekyll
├── assets/
└── downloads/
```

Do not upload a parent folder around these files. Delete or replace the current `index(1).html`; GitHub Pages requires the exact filename `index.html`.

Pages settings: main branch, /(root).
